GIT COMANDOS:

git init
inicia um novo projeto com git

git add .
adiciona todos os arquivos e os deixa prontos para serem commitados

git add
adiciona os arquivos da mesma maneira, mas de forma isolada.

git commit -m "mensagem no commit"
commit os arquivos no histórico

git log
mostra os ultimos commits, log de alterações

git status
como está o estado da nossa ramificação (branch)

git diff
mostra o que foi alterado
o que tem de alteração na ramificação

git merge
merge (mescla) as ramificações

git branch
mostra a branch atual

git checkout ""
muda pra branch mencionada

git branch -b
cria uma nova branch a partir da branch atual

git remote add
adiciona um novo repositório remoto

git push
manda nossas alterações locais para o repositório remoto, pra cada branch

git push origin master
envia a master atualizada para o repositório

git pull
pega as alterações do repositório remoto e joga para nossa máquina

git fetch
atualiza o novo histórico local de acordo com o histórico salvo no repositório
sincronização do local com o remoto ...
